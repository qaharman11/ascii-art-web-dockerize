# ASCII-ART-WEB-DOCKERIZE


### Usage:
```
#uses the Dockerfile to build a new container image
$ docker build -t web .

#to run application in the container use this command
$ docker run -p 8080:8080 -d web

#and open next link below:
localhost:8080/

```
